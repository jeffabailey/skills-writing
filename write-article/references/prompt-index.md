# Article Creation Prompt Index

Quick reference for all article creation prompts and when to use each framework.

---

## Documentation (Diataxis)

| Framework | Slug | Best For |
|-----------|------|----------|
| Tutorials | `diataxis-article-tutorials-create` | Learning by doing, guided lessons for beginners |
| How-to Guides | `diataxis-article-how-to-guides-create` | Task-focused steps for intermediate users |
| Reference | `diataxis-article-reference-create` | Lookup-oriented, scan-friendly factual content |
| Explanation | `diataxis-article-explanation-create` | Context, background, "why" questions |

Source: [Diataxis Framework](https://diataxis.fr/)

## Persuasion and Engagement

| Framework | Slug | Best For |

| Influence Pieces | `influence-pieces-article-create` | Social proof, authority, and framing |
